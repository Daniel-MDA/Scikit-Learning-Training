# Scikit-Learn — Laboratórios de Machine Learning

Laboratórios práticos de **machine learning** com [scikit-learn](https://scikit-learn.org/), desenvolvidos em Jupyter Notebook durante treinamento em 2023.

## Conteúdo

| Notebook | Tema |
|---|---|
| [Lab05_Regressao_Linear_02.ipynb](Lab05_Regressao_Linear_02.ipynb) | Regressão Linear |
| [Lab06_Regressao_Logistica_01.ipynb](Lab06_Regressao_Logistica_01.ipynb) | Regressão Logística |
| [Lab08_KNN_01.ipynb](Lab08_KNN_01.ipynb) | K-Nearest Neighbors (KNN) |

## Tecnologias

- Python
- scikit-learn
- Jupyter Notebook
- Pandas / NumPy / Matplotlib

## Como executar

```bash
git clone https://github.com/Daniel-MDA/Scikit-Learning-Training.git
cd Scikit-Learning-Training
pip install scikit-learn pandas numpy matplotlib jupyter
jupyter notebook
```

Ou abra os notebooks diretamente no [Google Colab](https://colab.research.google.com/).
